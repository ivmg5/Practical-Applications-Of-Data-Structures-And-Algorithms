# Proyecto Final de Estructuras de Datos y Algoritmos Fundamentales

## Descripción General

Este repositorio contiene los proyectos desarrollados durante el curso de Estructuras de Datos y Algoritmos Fundamentales en el [Tecnológico de Monterrey](https://tec.mx/). A lo largo de este curso, hemos trabajado en distintas actividades diseñadas para profundizar en conceptos clave de estructuras de datos y algoritmos, aplicando estos conocimientos para resolver problemas prácticos y mejorar la eficiencia de los programas.

## Tecnologías Utilizadas

- **Lenguaje de Programación:** C++
- **Entorno de Desarrollo:** Visual Studio Code, compatible con sistemas operativos Windows, macOS y Linux.
- **Control de Versiones:** Git & GitHub

## Estructura del Proyecto

El proyecto se divide en cuatro actividades principales, cada una enfocada en aspectos específicos de las estructuras de datos y los algoritmos:

### Actividad 1: Ordenamiento y Filtrado de Registros

Esta actividad se centró en el manejo de grandes volúmenes de datos, aplicando algoritmos de ordenamiento y técnicas de filtrado para procesar y analizar registros de una bitácora.

### Actividad 2: Manejo de Listas Doblemente Ligadas

Desarrollamos y optimizamos estructuras de datos complejas, específicamente listas doblemente ligadas, para gestionar eficientemente registros por dirección IP, implementando técnicas de ordenamiento avanzadas.

### Actividad 3: Optimización y Búsqueda en Listas Doblemente Ligadas

Continuamos con la optimización de las listas doblemente ligadas, enfocándonos en mejorar la eficiencia del ordenamiento y la búsqueda por rangos de direcciones IP, aplicando algoritmos adaptados a estas estructuras.

### Actividad 4: Hash Table y Lista Ligada para Resumen de Registros

Esta actividad involucró el desarrollo de una tabla hash y listas ligadas para procesar y resumir registros de bitácora, demostrando la aplicación de estas estructuras en el manejo y análisis de grandes conjuntos de datos.

## Instalación y Ejecución

Para ejecutar los programas incluidos en este repositorio, asegúrate de tener instalado un compilador de C++ que soporte C++11 o superior. Los pasos generales para compilar y ejecutar cada actividad son los siguientes:

1. Clona el repositorio en tu máquina local.
2. Abre una terminal o línea de comandos.
3. Navega hasta el directorio del proyecto.
4. Compila el código fuente utilizando un comando apropiado, por ejemplo, `g++ -o actividad1 actividad1.cpp` (ajusta el nombre del archivo de salida y la fuente según sea necesario).
5. Ejecuta el programa compilado, por ejemplo, `./actividad1`.
